# Evidencias

Guarda aquí capturas o archivos que demuestren tu proceso.

Evidencias sugeridas:

1. Consola con errores iniciales.
2. Registro del bug más importante.
3. Código antes y después de una corrección.
4. Sistema funcionando después de corregir.
5. Resultado de `tests.html`.
6. Validación de caso inválido.
7. Registro válido agregado a la tabla.
8. Retroalimentación recibida o publicada.
